# PR Workflow Commands

The PR lifecycle toolkit. Each subcommand has a dedicated file.

| Subcommand      | File              | Description                              |
|-----------------|-------------------|------------------------------------------|
| `/pr create`    | `pr_create.md`    | Cherry-pick commits into a clean PR      |
| `/pr review`    | `pr_review.md`    | Interactive file-by-file code review     |
| `/pr fix`       | `pr_fix.md`       | Diagnose and fix CI failures             |
| `/pr comments`  | `pr_comments.md`  | Address reviewer feedback one-by-one     |
| `/pr skeleton`  | `pr_skeleton.md`  | Extract architecture skeleton for review |
| `/pr mergetool` | `git_mergetool.md`| Intelligent timeline-based conflict resolution |
